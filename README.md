## ğŸ¯ How to Play

-   Player X starts first.
-   Players take turns clicking on empty squares.
-   The first player to align 3 marks **horizontally, vertically, or
    diagonally** wins.
-   If all squares are filled and no player wins, the game ends in a
    draw.
